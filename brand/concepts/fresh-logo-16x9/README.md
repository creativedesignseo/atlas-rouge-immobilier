# Atlas Rouge — Fresh Logo Concept 16:9

Independent logo exploration, intentionally separated from the existing
vectorized brand-book kit.

## Direction

- Editorial, quiet luxury, French-facing real-estate positioning.
- No touristic Marrakech cliches.
- `Atlas` in midnight ink, `Rouge` in terracotta.
- Abstract contour line as a restrained reference to the Atlas mountains and
  Marrakech's red earth, not a literal landscape illustration.
- French descriptor: `IMMOBILIER DE LUXE - MARRAKECH`.
