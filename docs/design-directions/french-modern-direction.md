# French Modern Direction

**Status:** active
**Date:** 2026-06-22
**Scope:** modern French real-estate brand, premium and contemporary, not Marrakech-led

## Intent

Create a modern real-estate visual language for France that feels serious, clean, and contemporary. The direction should read as a premium property brand rather than a template, with restrained color, sharp hierarchy, and a product-first layout.

## Palette

- `Porcelain` `#F7F7F0`
- `Carbon` `#161816`
- `Petrol` `#075B52`
- `Absinthe` `#C9F24A`
- `Brique` `#A8473D`
- `Linen Line` `#DDD8CC`

## Typography

- Headlines: `Manrope`, heavy weight, tight tracking, large scale
- Body: `Inter`, regular and medium only
- UI labels: `Inter`, uppercase sparingly, small and crisp

## Layout Rules

- White or porcelain background first
- One primary accent color per screen
- Real property imagery in the first viewport
- Search and key actions visible immediately
- No decorative hero cards, no artificial gradient scenes, no rounded demo blocks everywhere

## Visual Signals

- `Petrol` is the trust signal
- `Absinthe` is a rare highlight, not a theme color
- `Brique` is secondary and should stay limited
- Cards should stay flat and rectangular
- Borders should be thin and pale
